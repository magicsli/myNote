### npm  script ( npm脚本 )

​	概念:

​		npm 允许在`package.json`文件里面, 使用script字段定义脚本命令

​	常用脚本:

+ npm 脚本就相当于一个电器的使用说明书

+ npm 脚本运行使用 npm run 

+ npm 脚本可以简写,  但是只有特定几个
  + `npm start`  ==>  `npm run start`
  + `npm stop`  ==>  `npm  run  stop`
  + `npm  text`  ==> ` npm  run  test`
  + `npm  restart`  ==>  `npm run stop & npm run start`
